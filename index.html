Projetos em Destaque

  • ImageCompression API

    Esse projeto é uma API escrita em typeScript que tem como objetivo oferecer um serviço de compressão de imagens. As tecnologias usadas na sua contrução são: express para projetar a arquitetura da API, que segue um padrão voltado ao TDD(test driven development), jest e supertest para realizar as testagens, multer para facilitar o processo de upload e alocação das imagens e o sharp para fazer a compressão em si. O projeto tambem conta com o swagger ui para uma testagem de endpoints mais agradavel e pra fornecer a documentação completa da API.

  • BookStore API

    Esse projeto se trata de uma API escrita em typescript, que tem como objetivo gerenciar uma loja de livros. O projeto conta com o uso da arquitetura em camadas, usando os principios do SOLID e do TDD, além de realizar testes unitarios para uma melhor confiabilidade. No codigo fonte no github você terá acesso a documentação da API que foi totalemte documentada usando o Swagger UI.
